# C# ASP.NET MVC - Proof of Concept project

## Purpose
> To create an MVC project on Visual Studio to run a simple CRUD web application using:
 * Multitier architecture:
   * Data layer (Entity Framework)
   * Business layer
   * Service  layer (WCF)
   * Presentation layer:
     * Pure MVC
     * MVC + jQuery and DataTables
     * MVC + AngularJS
